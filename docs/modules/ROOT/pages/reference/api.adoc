= API Reference
:description: API documentation for OpenMind

== Core Classes

=== Mind Class

The central coordinator for goals and operations.

[source,cpp]
----
class Mind {
    // Goal management
    void addGoal(Goal& goal);
    void processGoals();
    
    // Generator management
    void registerGenerator(GoalGenerator& generator);
};
----
